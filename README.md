# EEG-Based Behavioral Prediction

This project builds a deep learning model to predict behavioral factors from EEG brain signals.

## Dataset

EEG Challenge Dataset

- 129 EEG channels
- Brain signal time series

## Model

Modified EEGNet architecture including:

- Convolutional layers for spatial feature extraction
- Multi-head attention
- Demographic feature integration

## Performance

RMSE ≈ **0.73**

## Tech Stack

Python  
PyTorch  
NumPy  
Pandas
